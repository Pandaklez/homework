Poetic Bot

Instructions and description

Бот берёт все стихи из НКРЯ с помощью лекси-граммтаического поиска
Команда /help вызывает подсказку
Команда /start запускает приветствие
Когда бот не находит искомое слово, он предлагает поискать другое
Если в стихотворении больше знаков, чем можно отправить в одном сообщении в телеграме, бот делит стихотворение на сообщения по несколько символов
Бот старается выделять искомое слово в стихотворении капсом
